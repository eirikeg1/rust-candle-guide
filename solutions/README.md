# Solutions

Reference implementations for all exercises. Each file is a drop-in replacement
for the corresponding `src/exercises/exNN_*.rs` module.

To check your work (or unblock yourself), copy a solution over the stub:

```bash
cp solutions/ex03_linear_regression.rs src/exercises/ex03_linear_regression.rs
```

Then run via CLI or GUI:

```bash
cargo run --bin ex03_linear_regression
cargo run --bin gui
```
